Quick reference for how to run a model in ADMB using BASH through the terminal

To open ADMB, open a terminal window and type 

admb

In bash, to change to directory ~ , run 

cd ./~

To list all folders/files in the directory type dir
To move up a directory type cd ..

To compile your model in safe mode, run

admb -s model

Finally, to run the model execute 

./model -anyRuntime command -anyOtherRuntime command

A list of available options can be accessed through the terminal by typing

./model -help

The default in ADMB is to use the PIN file with the same name as the model.TPL - to use a different PIN file than the default use runtime option 

-ainp differentFile.pin

Run Markov chain Monte-Carlo simulations

After you compile your model, as above, this is a two-step process. First use

./model -mcmc N -mcsave M

to run N total MCMC iterations with a thinning rate of M. The final step is to use

./model -mceval

which reads in the saved parameter vectors and calls the function for each - it is this final step which provides your output.

Trouble-shooting options

To see what your model does after a single function evalutaion and to see what the value of the OF is at first run using the PIN values use runtime option -maxfn 1. This can also be used for any other finite number (n) function evaluations by using -maxfn n.

Use -noest to turn estimation of all parameters off, and use -nohess to repress Hessian calculations which are unecessary in deterministic-test runs.
